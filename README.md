# AI Study Assistant 📚

An AI-powered web application that helps students ask doubts and get instant, easy-to-understand explanations. The app provides a simple and interactive interface to make learning easier and faster.

## Features

* Ask study-related questions
* Instant AI-generated explanations
* Simple and user-friendly interface
* Helps in quick learning and doubt solving

## Tech Stack

* HTML
* CSS
* JavaScript
* AI API / Backend

## Run the Project

```bash id="ecykgv"
npm install
npm start
```
